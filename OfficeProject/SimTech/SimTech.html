Designed and developed multiple VR-based educational applications for engineering students using the Unity Engine. Implemented over four interactive 3D projects featuring real-time simulations, physics-based interactions, optimized C# scripting, and immersive VR controls to enhance hands-on learning and conceptual understanding.

Porunai Museum Interactive Experience – Tirunelveli

Developed an interactive museum application for the Porunai Museum in Tirunelveli using Unreal Engine. Implemented gameplay systems using C++ and Blueprints to create engaging user interactions. Integrated Arduino hardware to capture real-world bellows press and release actions, synchronizing them with in-engine character animations. Also implemented drag-and-drop interaction systems to trigger animations and enhance the visitor learning experience.

Vaigai AI Chatbot – Interactive MetaHuman Assistant

Developed an AI-powered chatbot named Vaigai using Python and Unreal Engine, where Python processes user input and generates contextual responses. The system was integrated with Unreal Engine MetaHuman characters to deliver real-time voice responses with synchronized lip-sync animations. This project was created for the Tamil Nadu State Planning Commission, Chennai, enabling an interactive digital assistant experience through realistic MetaHuman interaction.

Unity Interactive Applications – Government Awareness Platforms

Developed optimized Unity-based applications and 2D gameplay systems for government initiatives including Districts of Distinction (DoD), Wings of Youth (WoY), and Wheel of Properties (WoP). These applications were integrated with the Tamil Nadu State Planning Commission, Chennai, creating interactive digital platforms that improved public awareness, learning accessibility, and user engagement through gamified experiences.
