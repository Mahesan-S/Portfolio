Introduction

Developed a VR Interactive House experience using Unity with the OpenXR SDK to create immersive room-based interactions. Configured XR Management with Android and Windows platform support to ensure cross-platform VR compatibility. Implemented interactive features using XR Interaction Toolkit and custom C# scripts. Focused on clean and optimized code by applying concepts like interfaces and class inheritance. This project is my first self-developed OpenXR project, built independently to explore VR interaction and system setup.
